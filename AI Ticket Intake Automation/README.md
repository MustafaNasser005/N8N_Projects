# AI Ticket Intake Automation

An end-to-end **AI-powered IT ticket intake and notification system** built using **n8n**, **Airtable**, **Google Gemini**, and **Gmail**.

This workflow automates how IT support tickets are collected, stored, summarized using AI, and communicated to both users and IT management.

## 🚀 Features

* Public IT request form (no login required)
* Automatic ticket creation in Airtable
* Email confirmation sent to the user
* AI-generated ticket summary using Google Gemini
* Automated email sent to IT management
* Ticket status tracking and email-sent flagging

---

## 🧩 Workflow Overview

The automation is split into **two main flows**:

### 1️⃣ Ticket Submission Flow (User Side)

1. User submits the **AI Ticket Intake Form**
2. A new record is created in **Airtable** with:

   * User name
   * Email
   * Issue description
   * Priority
   * Status set to `Pending`
3. A confirmation email is sent to the user with the Ticket ID
4. The Airtable record is updated to mark `Email Sent = true`

---

### 2️⃣ AI Notification Flow (IT Management Side)

1. **Airtable Trigger** listens for new Ticket IDs
2. Ticket data is sent to **Google Gemini**
3. Gemini generates a **structured JSON output** containing:

   * Email subject
   * Email content (clean text, bullet points)
4. An email summary is automatically sent to IT management

---

## 🧠 AI Behavior

The AI model is instructed to:

* Act as an IT support coordinator
* Generate concise, professional summaries
* Return **only valid JSON** (fully parseable by n8n)
* Avoid markdown or rich formatting
* Include all ticket details clearly

This ensures reliability and zero hallucination in production workflows.

## 📋 Airtable Schema

| Field Name     | Type           | Description                      |
| -------------- | -------------- | -------------------------------- |
| Ticket ID      | Formula / Auto | Unique ticket identifier         |
| User           | Text           | Full name                        |
| Email          | Email          | User email                       |
| Issue          | Long text      | Problem description              |
| Priority       | Select         | Low / Medium / High              |
| Status         | Select         | Pending / In Progress / Resolved |
| AI Summary     | Text           | Generated by Gemini              |
| Email Sent     | Checkbox       | User notification status         |
| Date Submitted | DateTime       | Submission timestamp             |

## ⚙️ Setup Instructions

1. Import the workflow JSON into **n8n**
2. Configure credentials:

   * Airtable Personal Access Token
   * Gmail OAuth2
   * Google Gemini (PaLM) API
3. Update email addresses for:

   * User confirmation
   * IT management notifications
4. Activate the workflow
